> **Správne konania na úseku ochrany prírody a krajiny**

+---+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
|   | V súlade s ustanovením § 82 ods. 7 zákona č. 543/2002 Z. z. o ochrane prírody a krajiny v znení neskorších predpisov **Okresný úrad Bratislava, odbor starostlivosti o životné prostredie,** **oddelenie ochrany prírody a vybraných zložiek životného prostredia** informuje o začatom správnom konaní, v ktorých môžu byť dotknuté záujmy ochrany prírody a krajiny chránené týmto zákonom                                                                                                                                                                                                                                                   |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | **Združenie s právnou subjektivitou**, ktorého predmetom činnosti najmenej jednej rok je ochrana prírody a  krajiny (§ 2 ods. 1) môže písomne oznámiť svoj záujem byť **zúčastnenou osobou** v začatom správnom konaní, pričom uvedie názov združenia, sídlo, identifikačné číslo organizácie, meno a priezvisko osoby oprávnenej konať v jeho mene a pripojí stanovy, ktoré preukazujú, že predmetom jeho činnosti je ochrana prírody a krajiny (§ 82 odsek 3 a 6), ak v minulosti nepodalo žiadosť účasť v bližšie nešpecifikovaných správnych konaniach, ktoré orgán ochrany prírody a krajiny v budúcnosti začne alebo v súčasnosti vedie. |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | Potvrdenie záujmu byť zúčastnenou osobou v začatom správnom konaní je možné oznámiť orgánu ochrany prírody a krajiny v pôsobnosti odboru starostlivosti o životné prostredie Okresného úradu Bratislava v lehote nie dlhšej ako 5 pracovných dní od zverejnenia upovedomenia o podaní žiadosti na elektronickom sídle správneho orgánu a to buď:                                                                                                                                                                                                                                                                                               |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | 1.  ústne do zápisnice v sídle správneho orgánu alebo,                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | 2.  písomne na adresu alebo osobným podaním na podateľni; Okresný úrad Bratislava, odbor starostlivosti o životné prostredie, oddelenie ochrany prírody a vybraných zložiek životného prostredia, Tomášikova 46, 832 05 Bratislava 3 alebo,                                                                                                                                                                                                                                                                                                                                                                                                    |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | 3.  elektronicky cez portál štátnej správy (slovensko.sk) s overeným elektronickým (ZEP) podpisom alebo,                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | 4.  mailom na adresu: <podatelna.ba@minv.sk> (avšak s povinnosťou do 3 pracovných dní prihlásenie potvrdiť podľa bodu 1 až 3),                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|   | Lehota začína plynúť dňom nasledujúcim po dni, kedy bola informácia o začatom konaní zverejnená.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
